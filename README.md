# gmcl\_rtmidi

This is a binary module for [Garry's Mod](https://gmod.facepunch.com/) that exposes realtime MIDI communication to the game's Lua environment through the [RtMidi library](https://caml.music.mcgill.ca/~gary/rtmidi/).

## Download

Compiled binaries are available from the [releases page on GitHub](https://github.com/inco-cc/gmcl_rtmidi/releases).
