# gmcl\_rtmidi

[![C++ 14](https://img.shields.io/badge/C++-14-blue)](https://en.cppreference.com/w/cpp/14)
[![RtMidi 6.0.0](https://img.shields.io/badge/rtmidi-6.0.0-blue)](https://www.music.mcgill.ca/~gary/rtmidi/)

gmcl\_rtmidi is a binary module for [Garry's Mod](https://gmod.facepunch.com/)
that exposes real-time MIDI communication to the Lua state through the
[RtMidi](https://www.music.mcgill.ca/~gary/rtmidi/) library.

## Download

Compiled binaries for gmcl\_rtmidi are available from
[Releases](https://github.com/inco-cc/gmcl_rtmidi/releases).

## Documentation

Documentation for gmcl\_rtmidi is maintained on the
[Wiki](https://github.com/inco-cc/gmcl_rtmidi/wiki).
