# gmcl\_rtmidi

This is a binary module for [Garry's Mod](https://gmod.facepunch.com/) that exposes realtime MIDI communication to the game's Lua environment through the [RtMidi](https://caml.music.mcgill.ca/~gary/rtmidi/) library.

## Download

Compiled binaries are available from the [releases](https://github.com/inco-cc/gmcl_rtmidi/releases) section on GitHub.
