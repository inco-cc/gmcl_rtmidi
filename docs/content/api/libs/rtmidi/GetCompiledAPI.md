---
title: GetCompiledAPI
---

```lua
table rtmidi.GetCompiledAPI()
```

## Description

Returns a list of compiled MIDI APIs that can be used by the host system.

## Returns

1. [table](https://wiki.facepunch.com/gmod/table)  
The list of compiled MIDI APIs. See the [rtmidi.API]({{% ref "api/libs/rtmidi/API" %}}) enumerations.
