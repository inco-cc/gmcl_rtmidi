---
title: GetAPIDisplayName
---

```lua
string rtmidi.GetAPIDisplayName(number api)
```

## Description

Returns the display name used for a MIDI API.

## Arguments

1. [number](https://wiki.facepunch.com/gmod/number) `api`  
The MIDI API. See the [rtmidi.API]({{% ref "api/libs/rtmidi/API" %}}) enumerations.

## Returns

1. [string](https://wiki.facepunch.com/gmod/string)  
The display name.
