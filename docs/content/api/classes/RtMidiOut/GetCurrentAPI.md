```lua
number RtMidiOut:GetCurrentAPI()
```

## Description

Returns the MIDI API currently being used.

## Returns

1. [number](https://wiki.facepunch.com/gmod/number)  
The current MIDI API. See the [rtmidi.API]({{% ref "api/libs/rtmidi/API" %}}) enumerations.
