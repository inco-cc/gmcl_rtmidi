---
title: gmcl_rtmidi
---

This is the official documentation for [gmcl_rtmidi](https://github.com/inco-cc/gmcl_rtmidi), a binary module for [Garry's Mod](https://gmod.facepunch.com/) that exposes realtime MIDI communication to the game's Lua environment through the [RtMidi library](https://caml.music.mcgill.ca/~gary/rtmidi/).
